# Language Tips (HTML)

Struttura pronta per GitHub Pages (static HTML).

## Come pubblicare su GitHub Pages
1. Crea un repo (es. `languages-tips`)
2. Carica tutti i file di questa cartella nella root del repo
3. Vai su Settings → Pages → Deploy from a branch → `main` / root
4. Apri l’URL GitHub Pages

## Link affiliati
- Sostituisci `https://www.babbel.com` con il tuo link tracciato (Awin/Impact).
- Mantieni il disclaimer visibile (footer + pagina Disclaimer).

## Email contatto
- Cambia `hello@example.com` in `contact.html` e `privacy.html`.
